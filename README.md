## Setup

### Install Node.js

Choose an installation method from <https://nodejs.org/en/download> and install the LTS version of Node.js (v24);

### Install PNPM

Enable the package manager, PNPM, via corepack (should still be installed by default in v24):

```sh
corepack enable pnpm
```

### Install Dependencies

Install all of the project's dependencies:

```sh
pnpm install
```

## Dev Preview

Preview the website within a dev environment (Prints a localhost URL, copy and paste it to your browser):

```sh
pnpm dev
```
